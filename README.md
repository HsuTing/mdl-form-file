# File for mdl-form

## Parameter

- `style` -> change style
- `label` -> name of file
- `isMultiple` -> if you use `isMultiple: true`, it can upload multiple files.
- `isNotRequire` -> if you use `isNotRequire: true`, it can be empty.

## Example

```
{   
  'id': 'component_id',
  'label': 'Label',
  'style': {'color': 'red'},
  'isMultiple': true,
  'isNotRequire': true
}
```
